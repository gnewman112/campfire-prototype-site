Purpose:

This site was developed with three priorities.

  1. Teach myself full stack web development
  2. Create a platform without ads or data collection/ sales, with content displayed in a larger format
  3. Demonstrate my ability to quickly learn distinctly different skills

Stages:

  1. Prototype
  2. Alpha Testing
  3. Beta Testing
  4. V.1 ...

This project is a hobby and currently in the prototype stage. In this stage, the focus is on adding features and social media providers.

If the alpha testing stage is reached, a small number of accounts will be opened, and focus will shift to identifying remaining bugs and adding requested features.

Development beyond the prototype stage is subject to demand. At a minimum I will add/support features I want, as I enjoy the advantages of building tools to control my social media feeds.
